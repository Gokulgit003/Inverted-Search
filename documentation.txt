Project Title: Inverted Search
Description: The project implements an inverted indexing system similar to a basic search engine. It processes multiple text files,
             extracts unique words, and stores them in a hash table structure. Each word node maintains a linked list of files 
             where it appears along with frequency counts. The system supports database creation, real-time search operations, 
             structured file saving, and database reconstruction. The implementation emphasizes efficient hashing, pointer-based 
             memory management, and modular C programming practices.
